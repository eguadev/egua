      <h2>Código que fala a sua língua</h2>
      <p>Em Égua, você programa com <code>se</code>, <code>senão</code>, <code>para</code>, <code>função</code> e <code>escreva</code>. Os conceitos são os mesmos de qualquer linguagem — variáveis, condições, laços de repetição e funções — mas escritos em português, sem a barreira do inglês.</p>
      <p>A base que você constrói aqui vale para todas as outras linguagens de programação.</p>
    </div>
    <div class="home-section-code">

```js
função classificar(nota) {
    se (nota >= 7) {
        retorna("Aprovado!");
    } senão {
        retorna("Continue estudando!");
    }
}

var notas = [8, 5.5, 9];

para (var i = 0; i < tamanho(notas); i = i + 1) {
    escreva(classificar(notas[i]));
}
```
